# 🏥 SehatSathi – Healthcare Web Application

## 📌 Overview  
SehatSathi is a full-stack healthcare web application designed to provide users with essential health-related services and information in a simple and accessible way. The platform focuses on improving user awareness, managing basic health data, and offering a smooth digital healthcare experience.

---

## 🚀 Features  
- 🔐 User Authentication (Login/Signup)  
- 📊 Health Data Management  
- 📄 Dynamic Content Rendering  
- 📱 Fully Responsive UI  
- ⚡ Fast and Interactive User Experience  
- 🌐 Accessible Healthcare Information  

---

## 🛠️ Tech Stack  
- **Frontend:** React.js, HTML, CSS, Bootstrap  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Version Control:** Git & GitHub  

---

## 🎯 Problem Statement  
Many people lack easy access to basic healthcare information and tools to manage their health data digitally. SehatSathi aims to bridge this gap by providing a centralized platform for essential healthcare support.

---

## 💡 Solution  
SehatSathi offers a user-friendly interface where users can securely manage their health-related information and access important healthcare resources anytime, anywhere.

---

- 📅 Appointment booking system
